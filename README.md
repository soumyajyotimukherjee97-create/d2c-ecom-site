# d2c-ecom-site

A full-stack D2C ecommerce website for a skincare brand, with a built-in order management system to handle customer purchases, track orders, and support internal operations.

## Overview

`d2c-ecom-site` is designed to serve as both a customer-facing storefront and an internal operational tool.

On the customer side, it enables users to browse skincare products, view product details, add items to cart, and place orders.

On the operations side, it includes an order management system that helps the brand manage incoming orders, monitor order status, and streamline fulfillment workflows.

This project is intended for D2C skincare businesses that want a unified platform for selling products online and managing post-purchase operations.
