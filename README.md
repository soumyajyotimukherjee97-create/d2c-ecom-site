# d2c-ecom-site

A full-stack D2C ecommerce website for a skincare brand, with a built-in order management system to handle customer purchases, track orders, and support internal operations.

## Overview

`d2c-ecom-site` is designed to serve as both a customer-facing storefront and an internal operational tool.

On the customer side, it enables users to browse skincare products, view product details, add items to cart, and place orders.

On the operations side, it includes an order management system that helps the brand manage incoming orders, monitor order status, and streamline fulfillment workflows.

This project is intended for D2C skincare businesses that want a unified platform for selling products online and managing post-purchase operations.

## Key Features

### Customer-facing ecommerce
- Product listing and product detail pages
- Category-based browsing
- Shopping cart functionality
- Secure checkout flow
- Order confirmation experience
- Responsive UI for desktop and mobile

### Order management system
- Centralized order dashboard
- View and manage customer orders
- Track order lifecycle from placement to fulfillment
- Update order statuses
- Access customer and order details
- Support operational workflows for order processing

### Brand and business needs
- Tailored for D2C skincare brands
- Supports product discovery and conversion
- Bridges frontend commerce with backend order operations
- Can be extended for inventory, shipping, returns, and customer support workflows

## Use Cases

This platform can be used to:
- Sell skincare products directly to consumers
- Manage incoming orders from a single system
- Improve order visibility for internal teams
- Create a branded ecommerce experience
- Build a foundation for future ecommerce and ops automation
